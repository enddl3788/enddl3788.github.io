신입생·방문객이 쉽게 학교를 탐색할 수 있는 모바일 지도 서비스

🗨️ 밀리톡 2025 공공데이터 활용 경진대회 출품작

기간: 2024.02 ~ 진행 중
카테고리: AI, 챗봇, 국방부, 공공데이터, KakaoTalk API
수상: 2025 국방부 경진대회 출품

프로젝트 개요
Militalk는 대한민국 국방부 오픈 API를 활용해 군인과 가족, 예비역에게 실시간 정보와 챗봇 서비스를 제공하는 Flask 기반 챗봇 API 서버입니다. 카카오톡, 슬랙, 웹 등 다양한 환경에 연동 가능하며 GPT-4.1 기반 자연어 질의응답도 지원합니다.

✨ 핵심 기능
  • 📦 군 복지/병원/식단/PX 정보 제공
  • 🎫 TMO, 병사 할인, 기숙사 정보 지원
  • 📅 날짜 인식: "오늘", "내일", "2024년 6월 18일" 등 해석
  • 🤖 GPT-3.5 상담/질의 API 지원
  • 📚 군사 용어/복무 계산기 기능 내장
🛠️ 기술 스택
  • 백엔드: Python 3, Flask
  • 외부 API: 국방부 OpenAPI, OpenAI API (GPT-4.1)
  • 데이터 처리: requests, json, datetime, python-dateutil
  • 비동기: threading (OpenAI 응답 병렬 처리)
  • 배포: 개인 서버 운영
📘 API 라우트 예시
  • /welfare: 군 복지 휴양시설
  • /tmo_info: TMO 안내소
  • /salary_info: 병사 월급 정보
  • /question: GPT 질의응답 (자연어)
성과 및 성장
  • 공공데이터-실무 직접연동, 위기발화 탐지 자동실행 경험
  • 실제 장병/가족들의 반응 피드백 수집 및 서비스 개선
