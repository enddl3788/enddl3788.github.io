기간: 2024.02 ~

수상: 2025 국방부 경진대회 출품

프로젝트 개요
군인·가족·예비역 대상 실시간 군생활 맞춤형 챗봇(AI+오픈API 기반)을 개발. 카카오톡/슬랙/웹 등에서 사용 가능하며, GPT-3.5와 국방부 공공데이터를 함께 사용해 기존 검색보다 쉽고 정확한 답변과 위기상황 응원 안내가 가능.

핵심 기능
  • 국방 오픈 API 실시간 조회(복지, 월급, 식단, 병원 등 10+종)
  • GPT-3.5 통합 자연어 질의, 감정 대응(위기 발화 시 1303 안내)
  • 사용자 입력 자유화(자연어 날짜/계급/분야 자동 파싱, 이모티콘 안내 등)
사용 기술
  • Python, Flask, OpenAI, 국방부 OpenAPI, RESTful API 등
  • 카카오챗봇, 슬랙 연동, 단독 백엔드 설계/구현
성과 및 성장
  • 공공데이터-실무 직접연동, 위기발화 탐지 자동실행 경험
  • 실제 장병/가족들의 반응 피드백 수집 및 서비스 개선
